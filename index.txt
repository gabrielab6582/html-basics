HTML Basics

The "frontend" of a webpage consists of:
HTML = HyperText Markup Language
content
CSS = Cascading Style Sheet
style
JS = Javascript
interactivity

Vocabulary
elements have the following syntax: &lt;tag attribute="value"&gt;content&lt;/tag&gt;





heading: &lt;h1&gt;&lt;/h1&gt; through &lt;h6&gt;&lt;/h6&gt;
horizontal rule (line): &lt;hr&gt;
paragraph: &lt;p&gt;&lt;/p&gt;
ordered list: &lt;ol&gt;&lt;/ol&gt;
unordered list: &lt;ul&gt;&lt;/ul&gt;
list item: &lt;li&gt;&lt;/li&gt;
form: &lt;form&gt;&lt;/form&gt;
div (container): &lt;div&gt;&lt;/div&gt;
